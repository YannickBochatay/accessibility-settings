<access-settings>

Démarrage rapide

Développeur-se

Ajouter l'appel du script dans la balise <head> de votre page html : 

<script
  src="https://cdn.jsdelivr.net/npm/access-settings"
  type="module"
></script>

Puis insérer le composant en fin de balise <body> :

<access-settings all></access-settings>

Et le tour est joué ! Pour une utilisation plus avancée, vous pouvez parcourir la suite de cette documentation.

Internaute (βeta)

Vous pouvez utiliser le composant <access-settings> en tant que bookmarklet.

Pour cela, faites glisser le bouton suivant jusqu'à votre barre de favoris :

Sur n'importe quelle page web, vous pourrez cliquer sur ce favori pour configuer vos préférences d'accessibilité.

Avertissement : selon la politique de sécurité de la page visitée ainsi que la qualité du code css, il se peut que le composant ne puisse pas se charger ou ne fonctionne pas correctement.

Types d'installation

CDN

Comme indiqué plus haut, l'utilisation d'un serveur CDN est la façon la plus simple et la plus rapide d'installer le composant. 
<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta charset="UTF-8">
    <title>My accessible page</title>
    <script
      src="https://cdn.jsdelivr.net/npm/access-settings"
      type="module"
    >
    </script>
  </head>
  <body>
    <access-settings all></access-settings>
  </body>
</html>

NPM

Vous pouvez aussi installer le composant par NPM. 
npm install access-settings
<script src="node_modules/access-settings/dist/index.js"></script>

Téléchargement direct

Ou encore télécharger directement le fichier sur GitHub.

Options d'utilisation

all

Pour afficher toutes les options, utiliser l'attribut all.

<access-settings all></access-settings>

dyslexic-font

Pour activer l'option police dyslexie, utiliser l'attribut dyslexic-font.

La police utilisée est Open Dyslexic depuis le serveur CDN|Fonts.

<access-settings dyslexic-font>
</access-settings>

invert-colors

Pour activer l'option d'inversion des couleurs, utiliser l'attribut invert-colors.

<access-settings invert-colors>
</access-settings>

contrast

Pour activer le réglage du contraste, utiliser l'attribut contrast.

<access-settings contrast>
</access-settings>

font-size

Pour activer le réglage de la taille du texte, utiliser l'attribut font-size.

<access-settings font-size>
</access-settings>

line-height

Pour activer le réglage de la hauteur de ligne, utiliser l'attribut line-height.

<access-settings line-height>
</access-settings>

combinaison

Vous pouvez bien sûr utiliser une combinaison de ces options.

<access-settings contrast font-size line-height>
</access-settings>

side="left"

Par défaut le composant est placé sur le bord droit de l'écran. Pour le positionner à gauche, utiliser l'attribut side="left" 

<access-settings all side="left">
</access-settings>

rounded

Pour que le contour de l'icône soit arrondi, utiliser l'attribut rounded

<access-settings all rounded>

lang

La langue est déterminée par l'attribut lang de l'élément racine html.

Les langues disponibles sont français (fr), anglais (en) et espagnol (es).

important

Pour outrepasser les règles css existantes, ajouter l'attribut important.

Attention, cette méthode n'est pas conseillée, elle permet juste le bon fonctionnement du composant dans le cas où le css existant ne suit pas les bonnes pratiques et est difficilement modifiable. 

<access-settings all important>

Personnalisation

Ajouter une langue

Vous pouvez ajouter la langue de votre choix de la façon suivante.

<!DOCTYPE html>
<html lang="oc">
import { AccessSettings } from "https://cdn.jsdelivr.net/npm/access-settings";

AccessSettings.languages.oc = {
  "dyslexic-font": "Poliça disléxica",
  "inverted-colors": "Colors invertidas",
  "contrast": "Contraste",
  "font-size": "Talha de poliça",
  "line-height": "Nautor de linha",
  "reset": "Reïnicializar",
  "close": "Tampar"
};

Position

Vous pouvez définir le positionnement du composant.

accessibility-settings {
  position:fixed;
  top:5px;
  right:5px;
}

Si l'attribute side est défini à left, il vous faudra définir la propriété css left plutôt que right.

Style de l'icône

Vous pouvez personnaliser le style de l'icône en css 
access-settings::part(icon) {
  background-color: brown;
  fill:white;
}

Changer d'icône

<access-settings all>
  <span slot="icon">⚙︎</span>
</access-settings>

Style des éléments

Les pseudo-classes ::part suivantes vous permettent de personnaliser entièrement le style du composant :

  • details : élément html <details> global
    • summary : élément html <summary>
      • icon : icône <svg>
    • form : élément html <form>

Ajouter des options

Vous pouvez ajouter des options de la façon suivante : 
<access-settings all>
  <div slot="option">
    <label>
      <input type="checkbox" id="custom-input">
      Une nouvelle option
    </label>
  </div>
</access-settings>
const input = document.querySelector("#custom-input");
input.addEventListener("change", () => {
  if (input.checked) alert("nouvelle option cochée");
})

Évènements

change

Vous pouvez ajouter des écouteurs sur l'évènement change.

La propriété detail de l'évènement contient les propriétés :

  • prop : nom de la propriété modifiée
  • value : nouvelle valeur
  • preferences : valeurs actuelles de toutes les préférences 
const access = document.querySelector("access-settings");

access.addEventListener("change", e => {
  const { prop, value, preferences } = e.detail;

  alert(`Property ${prop} has changed to ${value}.
  Preferences are now ${JSON.stringify(preferences)}.`);
})
